## Question 1
**Trouver le maximum et le minimum d'un tableau**

Écrivez une fonction qui retourne le plus grand et le plus petit élément d’un tableau d’entiers

**Par exemple :**
```
Entrée : {4, 45, 73, 59, 33}
Sortie : max = 73, min = 4
```

## Question 2
**Inverser un tableau**

Écrivez un programme pour inverser les données d'un tableau sans crée un nouveau tableau

**Par exemple :**
```
Entrée : {1, 2, 3. 4}
Sortie : {4, 3, 2, 1}
```

## Question 3
**Vérifier si un tableau est trié**

Écrivez un programme pour vérifier si le tableau est trié ou pas

**Par exemple :**
```
Entrée : {1, 2, 2. 3, 5}
Sortie : true

Entrée : {2, 2, 5. 1, 3}
Sortie : false
```

## Question 4
**Trouver l'élément le deuxième plus grand**

Écrivez une fonction Go qui retourne le deuxième plus grand nombre du tableau.

**Par exemple :**
```
Entrée : {10, 5, 20, 8}
Sortie : 10
```

## Question 5

Écrivez une fonction qui retourne le nombre de voyelles dans une chaine.

**Par exemple :**
```
Entrée utilisateur : str = "Hello World"
Sortie : 3
```
